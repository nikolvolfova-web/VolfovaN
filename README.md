# Analysis of Mitochondrial Ultrastructure and Morphology 

Biochemical and genetic scientist interested in bioimage analysis, microscopy, toxicology, genetics, and open-source scientific tools.

## Scientific software and image analysis

* [Analysis of Mitochondrial Ultrastructure and Morphology](https://github.com/LMCF-IMG/Analysis_Mitochondrial_Ultrastructure_and_Morphology)
  Contributed to the development and application of an image analysis workflow for mitochondrial ultrastructure and cristae morphology in electron microscopy data. My contribution included image analysis cooperation, manual annotation, and preparation of training image datasets for deep-learning-based segmentation models.

## Current work

I develop custom ImageJ- and Python-based image analysis methods for microscopy datasets, with an emphasis on reproducibility, accessibility, and modifiability.

I am currently preparing an open-source workflow for object-based organelle colocalization analysis and classification by percentage overlap.
